# 🚀 Docker to Kubernetes DevOps Projects

A complete hands-on DevOps learning repository covering real-world projects using Docker, Docker Compose, microservices architecture, caching, reverse proxy, Kubernetes, and CI/CD pipelines.

---

## 📌 Overview

This repository contains **7 practical DevOps projects** designed to build strong fundamentals and production-ready skills.

Each project focuses on real-world scenarios including containerization, service communication, performance optimization, and deployment automation.

---

## 🧠 Tech Stack

* Docker & Docker Compose
* Nginx (Reverse Proxy)
* Node.js / React
* PostgreSQL
* Redis (Caching)
* Kubernetes (K8s)
* GitHub Actions (CI/CD)
